# Портал «Учусь.РФ»

Демонстрационный экзамен, вариант В5 (09.02.07). Django-приложение для записи на онлайн-курсы.

## Запуск

```bash
py -m pip install -r requirements.txt
py manage.py migrate
py manage.py seed_courses
py manage.py runserver
```

Откройте http://127.0.0.1:8000/

## Учётные данные

**Администратор панели заявок:** логин `Admin26`, пароль `Demo20` — страница `/admin/login/`

**Пользователь:** регистрация на `/register/` (логин: латиница и цифры, мин. 6 символов; пароль: мин. 8 символов)

## Страницы

| URL | Описание |
|-----|----------|
| `/register/` | Регистрация |
| `/login/` | Вход |
| `/cabinet/` | Личный кабинет, заявки, отзывы, слайдер |
| `/application/new/` | Оформление заявки |
| `/admin/` | Панель администратора |

## ER-модель

- **User** — логин, пароль, ФИО, телефон, e-mail
- **Course** — курсы (повышение квалификации, переподготовка, охрана труда)
- **Application** — заявка (курс, дата, оплата, статус)
- **Review** — отзыв к заявке (после смены статуса админом)

Статусы заявки: «Новая» → «Идет обучение» → «Обучение завершено».

**Способы оплаты:** предоплата по QR-коду, оплата картой МИР, постоплата в офисе организации.

**Стиль:** палитра #007bff / #0d47a1 / #ced4da / #dee2e6, шрифт Inter.
